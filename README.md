# Quick Tally for Wear OS
A standalone quick tally app for Wear OS
## Features
- Watch tile
- Watch face complication
- Ongoing quick tally
- Ambient mode aware
- Supports dynamic theme and user selected theme color
- Standalone so no need for phone
- No ads
- Open source
